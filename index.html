Projects

Take a look at all the projects I've worked on since becoming a game development student.

This includes internship projects and are not ordered chronologically.

FlavorQuest Mobile

The project I might be the most proud of. A 2D top-down adventure featuring a piece of mutant jelly. This was my final project before the internship at Letink Design. All assets by me.

FlavorQuest Mobile

Before making the mobile version of FlavorQuest, I came up with the idea for a small platformer game first. Here, the game was a 2D sidescrolling adventure, featuring combat. All assets by me.

KPI Dashboard

My main assignment during my internship at Letink Design was to make a KPI Dashboard where the company's productivity an be overseen easily. I made the Unity UI functionality but did not design the visuals.


(Note: The screenshot is using placeholder variables)

VR Teleport package (Oculus Quest)

During the internship at Letink Design, I made a custom VR-Teleportation package that makes it easy to navigate through a VR-environment without getting dizzy.

VR Environment Setup

The VR Teleport package was created for this project. My task was to set materials and generate lighting on the given assets. In addition to that, the client needed to be able to move through it in VR.

Car Theory Simulation

The main assignment for my internship at GameLab Oost was to create an app that teaches the necessary car theory to mentally challenged people.


This was a serious game where the user could earn coins by answering questions correctly. These coins could be used to "pimp their ride"

Obstacle Course

After my internship at GameLab Oost was over, I spent a few additional weeks working on 2 minigames. Obstacle Course being one of them.


This game is controlled on an iPad with touch-controls. It bears a striking resemblance to a classic video game.

HamsterMania

For Saxion's school project 4: "Final Approach", I programmed this game in Saxion's C#-based 'GXP-Engine'.

The game features a Hamster in a ball that the player launches through an obstacle course. There's lots of small details and charm to be found in this game.

This Blows (game)

Saxion's 3rd school project "Lift-Off". During this project, I was also the only engineer and worked with a team of students to create a game about a bazooka-like leafblower weapon.

As with HamsterMania, this game was made in Saxion's GXP engine.

DownHill

A fun little mobile game I made to improve in the Unity Engine. In this game, I focused a lot on UI animations and transitioning between scenes.
