FPSepController Unity Package

A First-Person Rigidbody-based player controller package for the Unity Engine.

Both Simple-Camera and Cinemachine versions allow for moving, sprinting, crawling and jumping by dragging and dropping the prefabs into any scene.
Also pretty flexible, as I've used this package as a base for the controller in BOWLOUT.
